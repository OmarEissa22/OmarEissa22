##  Featured Project

### [CareQueue — AI-Powered GP Triage & Booking System](https://github.com/OmarEissa22/ai-gp-booking-system)
> Final Year Project · TypeScript · GPT-4o · LangChain · tRPC · Supabase · React

A full-stack web application where patients describe symptoms, GPT-4o triages urgency and routes them to the correct clinician, and a slot is automatically booked — no human intervention required.

**Key engineering highlights:**
- 3 integrated AI pipelines (pre-screening safety filter → triage engine → LangChain conversational assistant)
- Dual-mode chatbot: patient-facing (appointments, triage history, profile) + admin-facing (system stats, all triage results)
- Atomic slot reservation to prevent race conditions · JWT role-based access control · append-only audit log
- 10/10 functional test pass rate · validated by 7 users including health domain experts
